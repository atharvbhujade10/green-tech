Sustainability Checker — Streamlit App

This project is a machine learning–based Sustainability Checker built using Streamlit, Pickle, and NumPy.
The app takes environmental and efficiency-related inputs and predicts whether a system is Sustainable or Not Sustainable using a pretrained model (lrmodel_sustainable.pkl).

Features

Simple and interactive Streamlit web interface

Accepts the following input parameters:

Carbon Emission

Energy Output

Renewability Index

Cost Efficiency Index

Uses a pre-trained Logistic Regression model

Provides instant sustainability prediction
